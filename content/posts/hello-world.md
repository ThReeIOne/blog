## 为什么搭建个人博客

在信息爆炸的时代，拥有一个属于自己的写作空间显得尤为重要。个人博客不仅是技术积累的载体，更是思考与表达的出口。

## 技术选型

这个博客使用了以下技术栈：

- **Next.js 15** — React 全栈框架，支持静态生成
- **Tailwind CSS v4** — 原子化 CSS，快速构建界面
- **Markdown** — 纯文本写作，专注内容

## 一段示例代码

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```
